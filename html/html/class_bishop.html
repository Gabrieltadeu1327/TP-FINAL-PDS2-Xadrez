Detailed Description

Define os possíveis movimentos.

Dentro da subclasse Bishop, que herda de Piece, construímos o bispo recebendo sua cor, determinamos uma matriz de possíveis movimentos recebendo seu ponto atual.

void Bishop::def_possible_movements ( int *  spot)
overridevirtual

Função sobrescreve a função, da classe mãe, que define possíveis movimentos para o bispo (preto ou branco).

Recebendo a posição atual do bispo, a função altera a matriz de possíveis movimentos, declarada na classe 'piece', colocando '1' para movimento possível e '0' para movimento impossível, note que ainda não existe a verificação de validez do movimento. Ocorre sobrescrição da função virtual declarada em piece.

Parameters
spotponto inteiro atual da peça, necessário para determinar seus possíveis movimentos futuros.
Returns
void.
